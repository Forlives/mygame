# Bacteria War

A real-time HTML5 strategy game where players command bacteria colonies in microscopic warfare.

## Gameplay

- 3D Procedural World - Three.js rendered microscopic battlefield
- Real-time Strategy - Command bacteria units, capture territory
- Multiplayer - Supabase-powered real-time multiplayer
- Audio System - Dynamic sound effects and background music

## Tech Stack

- Rendering: Three.js (3D WebGL)
- Backend: Supabase (Realtime + Auth + Database)
- Deployment: Vercel

## Project Structure

- index.html - Game entry point
- js/main.js - Game loop and initialization
- js/battle.js - Combat system
- js/scene3d.js - Three.js 3D rendering
- js/procedural.js - Procedural world generation
- js/audio.js - Audio system
- js/supabase.js - Multiplayer backend
- js/config.js - Game configuration
